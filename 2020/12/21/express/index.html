Linux下部署Node.js和Express框架

下载putty、WinSCP

输入账号密码，登录root

输入pwd会发现此时默认在/root
cd /root

下载node和mysql的Linux镜像
wget https://npm.taobao.org/mirrors/node/v14.15.2/node-v14.15.2-linux-x64.tar.xz
wget https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.32-linux-glibc2.12-x86_64.tar.gz
（如果太慢了就在电脑上复制链接下载好，用WinSCP传过去）

解压
tar -xvf ./node-v14.15.2-linux-x64.tar.xz
tar -xzvf ./mysql-5.7.17-linux-glibc2.5-x86_64.tar.gz

为了环境卫生，删除已解压的压缩包
rm -rf ./node-v14.15.2-linux-x64.tar.xz
rm -rf ./mysql-5.7.17-linux-glibc2.5-x86_64.tar.gz

移动并修改文件名
mv ./mysql-5.7.17-linux-glibc2.5-x86_64 /usr/local/mysql

cd /usr/local
可以看到在 /usr/local/ 目录下，有个mysql文件夹，然后开始mysql用户组和mysql用户，添加完使用groups mysql查看是否添加成功
groupadd mysql
useradd -r -g mysql mysql
groups mysql

修改mysql目录拥有者为刚建立的mysql用户
cd ./mysql
chown -R mysql:mysql ./

MySQL 初始化
./scripts/mysql_install_db –user=mysql –basedir=/usr/local/mysql –datadir=/usr/local/mysql/data
修改mysql目录拥有者为root用户，修改data目录拥有者为mysql
chown -R root:root ./ && chown -R mysql:mysql data
启动mysql ,修改root密码，成功登陆
./support-files/mysql.server start
./bin/mysqladmin -u root password ‘00000000’
./bin/mysql -h127.0.0.1 -uroot -p00000000

配置环境变量
ln -s /root/node-v14.15.2-linux-x64/bin/node /usr/local/bin/node
ln -s /root/node-v14.15.2-linux-x64/bin/npm /usr/local/bin/npm

查看环境配置是否完成
node -v
npm -v

注：低版本的npm下载依赖比较慢，若下载速度慢，可以使用以下命令可以将npm升级至最新版本
npm install -g npm

npm install cnpm -g –registry=https://registry.npm.taobao.org

全局安装express
cnpm install -g express
cnpm install -g express-generator

安装pm2
执行如下命令来安装 pm2
npm install pm2 -g
安装完成后，执行命令，为我们的 pm2 添加开机自启动
pm2 startup systemd
pm2 save –force
设置为全局，创建软链
ln -s /root/node-v14.15.2-linux-x64/lib/node_modules/pm2/bin/pm2 /usr/local/bin
创建一个 express 站点

接下来，我们来创建一个 express 站点， 执行如下命令，来初始化一个新的以 cloud 为名的应用，
cd /home/
express cloud
初始化完成后，进入到目录中，安装对应的依赖
cd cloud
cnpm install
测试express
执行如下命令，来启动 Express ，进行测试
DEBUG=cloud:* npm start
此时，我们可以打开浏览器，访问 http://111.231.56.173:3000 就可以访问默认的 Express 界面的内容。
