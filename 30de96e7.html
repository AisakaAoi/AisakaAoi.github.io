进程与线程的区别

  1. 进程是资源分配的基本单位；线程是调度的基本单位。

  2. 切换进程比切换线程会有更大时间开销。

  3. 一个进程有多个线程。一个线程对应一个进程。

  4. 进程之间相互独立，线程之间共享。

堆和栈的区别

  • 栈由操作系统自动分配和释放

  • 堆由程序员申请和释放。

内核态和用户态的划分与切换

  1. 运行特权级别不同。用户态只有最低的特权级。内核态有比较高的。

  2. 用户态程序不能直接访问内核数据结构和程序。

  3. 用户态到内核态：系统调用、异常、外设中断。

虚拟地址怎么映射到物理地址

  1. 虚拟地址 = 页号 + 页内地址

  2. 页号映射到某个物理块，然后加上偏移量，就是页内地址。

  3. 映射的话，首先每个进程会有一个寄存器，专门存储页表始址和页表长度，通过计算页号是否超过页表长度，如果超出长度，则报错。如果没有超出长度，就将页号加上页表始址得到页表项的物理地址，然后访存得到对应的主存块号。主存块号和页内地址合起来就是我们要找的物理地址。

  4. 这可能会有一个快表的结构。就是页号直接映射到物理块号。如果存在，就直接得到了物理块号，如果不存在，就执行刚刚说的操作。如果快表满了，就按照一定算法替换一个快表项。

大小端

  • 大端：手写顺序，LSB 在高地址

  • 小端：LSB 在低地址

用 union 判断，他们会对齐于低地址。然后给 char 赋值，第一个字节（从低地址开始数）会是 0x01 。

然后如果是大端，就 MSB 在低地址，int 就不等于 1。

如果是小端的话， LSB 在低地址， int 等于 1 。
