2.1 API = 数据“外卖服务”

API（应用程序接口）就像餐厅的外卖服务：

  • 你下单（发送请求URL）
  • 厨房做饭（服务器处理）
  • 外卖小哥送上门（返回JSON数据）

免费财经API推荐：

  • 汇率：https://api.exchangerate-api.com/v4/latest/USD（免费，无需注册）
  • 国内汇率：https://www.banhou.com/api/v1/exchange_rate（模拟）
  • 黄金价格：https://api.gold-api.com/price/XAU（模拟）
